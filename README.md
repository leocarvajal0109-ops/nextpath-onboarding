# NextPath — Onboarding de agentes

Guía de onboarding interactiva para agentes de NextPath (seguros de vida). Es una página web autónoma de un solo archivo (`index.html`): todo el CSS, JavaScript e imágenes van embebidos, sin dependencias externas.

## Cómo verla
- En vivo (GitHub Pages): https://leocarvajal0109-ops.github.io/nextpath-onboarding/
- Local: descarga `index.html` y ábrelo en cualquier navegador.

## Contenido
Licencia, producto (IUL · FEX Whole Life · Term), stack diario (GoHighLevel + Aloware), proceso de 7 pasos, manejo de objeciones y scripts básicos.
